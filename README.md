# Nixia - Hybrid Financial Platform for Entrepreneurs

**Nixia** is a comprehensive financial platform designed specifically for African entrepreneurs who want to streamline company creation, accounting, tax filings, and compliance management.

## Overview

Built with modern web technologies, Nixia provides:

- **Company Registration Management**: Track and manage your company registration process from start to finish
- **Document Management**: Centralized storage and organization of all business documents
- **Tax & Accounting**: Automated tax calculations and accounting management
- **Compliance Tracking**: Stay on top of regulatory requirements and deadlines
- **AI Assistant**: Get intelligent support for business decisions
- **Professional Dashboard**: Beautiful, intuitive interface for business management

## Technology Stack

- **Frontend**: Next.js 16, React 19, TypeScript
- **UI Components**: shadcn/ui with Tailwind CSS
- **Icons**: HugeIcons
- **3D Graphics**: Three.js
- **Styling**: Tailwind CSS with PostCSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the platform.

## Building for Production

```bash
npm run build
npm start
```
